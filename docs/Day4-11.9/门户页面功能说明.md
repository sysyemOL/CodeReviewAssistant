# 门户页面功能说明

## 功能概述
创建了一个精美的AI应用产品门户页面，作为"Liu Shuo's 雷客AI应用研发挑战赛作品"的展示入口。

## 设计特点

### 1. 视觉设计
- **色调**：蓝白配色主题，清爽优雅
- **动态背景**：与ReviewWorkspace相同的动态渐变背景效果
  - 浅蓝、爱丽丝蓝、天蓝等多色渐变
  - 15秒循环动画，流畅自然
- **湖水镜面效果**：页面中间的水平线将内容分为上下两部分，下方为反射区域
  - 反射内容垂直翻转 (`scaleY(-1)`)
  - 渐变蒙版效果 (`mask-image`)
  - 40%透明度营造真实水面倒影

### 2. 页面结构

#### 顶部标题
```
Liu Shuo's 雷客AI应用研发挑战赛作品
```
- "Liu Shuo"：蓝色高亮，大字号（36px）
- "雷客AI应用研发挑战赛作品"：浅蓝色（#5a9fd4）
- 文字阴影效果增强层次感

#### 产品展示区
**左侧：AI代码Review助手**
- ✅ 蓝色渐变圆形图标
- 代码符号 `</>`
- 标题："AI代码Review助手"
- 描述："智能代码审查 · 实时对话 · 多语言支持"
- 状态徽章："已上线"（蓝色渐变）
- 可点击，跳转到 `/workspace`

**右侧：AI会议Summary助手**
- 📝 浅灰色圆形图标
- 会议记录图标
- 标题："AI会议Summary助手"
- 描述："会议记录 · 智能总结 · 任务提取"
- 状态徽章："待开发"（浅蓝色边框）
- 禁用点击

#### 页脚
```
© 2025 Liu Shuo - 雷客AI应用研发挑战赛
```

### 3. 交互效果

#### 悬停效果
- 产品卡片：
  - 向上移动 15px
  - 放大 5%
  - 增强阴影效果
  - 边框颜色加深
- 产品图标：
  - 放大 10%
  - 暂停悬浮动画

#### 动画效果
- **产品图标**：持续悬浮动画（上下8px位移，3-4秒循环）
- **背景渐变**：15秒循环渐变动画
- **过渡动画**：所有交互使用 0.4s 缓动过渡

### 4. 技术实现

#### 核心文件
- **组件**：`vue3-front/vue-project/src/views/PortalPage.vue`
- **路由**：`vue3-front/vue-project/src/router/index.js`

#### 关键技术点

**动态渐变背景**
```css
background: linear-gradient(-45deg, 
  rgba(225, 240, 255, 0.98) 0%,
  rgba(240, 248, 255, 0.98) 25%,
  rgba(230, 245, 255, 0.98) 50%,
  rgba(240, 250, 255, 0.98) 75%,
  rgba(235, 245, 255, 0.98) 100%
);
background-size: 400% 400%;
animation: gradientShift 15s ease infinite;
```

**水面镜面效果**
```css
.water-reflection {
  transform: scaleY(-1);  /* 垂直翻转 */
  opacity: 0.4;
  mask-image: linear-gradient(to bottom, rgba(0,0,0,0.6) 0%, transparent 100%);
  pointer-events: none;   /* 禁用交互 */
}
```

**水面线**
```css
.water-line {
  height: 2px;
  background: linear-gradient(to right, 
    transparent 0%,
    rgba(100, 180, 255, 0.3) 20%,
    rgba(100, 180, 255, 0.5) 50%,
    rgba(100, 180, 255, 0.3) 80%,
    transparent 100%
  );
  box-shadow: 0 0 20px rgba(100, 180, 255, 0.3);
}
```

**SVG 图标渐变**
```html
<linearGradient id="blueGradient">
  <stop offset="0%" style="stop-color:#409EFF;stop-opacity:1" />
  <stop offset="100%" style="stop-color:#66b1ff;stop-opacity:1" />
</linearGradient>
```

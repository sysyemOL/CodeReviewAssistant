# 返回门户按钮功能说明

## 功能概述
在AI代码Review助手工作区页面的左下角添加了一个圆形返回按钮，用户可以点击快速返回到产品门户页面。

## 功能特点

### 1. 位置与样式
- **位置**：位于左侧边栏（sidebar）的底部中央
- **形状**：圆形按钮
- **图标**：Home 图标（HomeFilled）
- **颜色**：蓝色渐变背景（#409EFF 到 #66b1ff）
- **阴影**：柔和的蓝色阴影效果

### 2. 交互效果
- **悬停**：按钮放大 10%，阴影增强
- **点击**：按钮缩小至 95%，提供视觉反馈
- **功能**：点击后导航到门户页面（`/`）

### 3. 响应式适配
- **展开状态**：按钮位于侧边栏底部中央
- **收缩状态**：按钮位置保持在底部中央，自适应收缩后的侧边栏宽度
- **层级**：z-index: 100，确保在最上层显示

## 技术实现

### 组件修改
**文件**：`vue3-front/vue-project/src/views/ReviewWorkspace.vue`

#### HTML 结构
```vue
<!-- 返回门户按钮 -->
<div class="back-to-portal">
  <el-button 
    :icon="HomeFilled" 
    circle
    class="portal-btn"
    @click="goBackToPortal"
    title="返回门户"
  />
</div>
```

#### 脚本逻辑
```javascript
// 导入图标和路由
import { HomeFilled } from '@element-plus/icons-vue'
import { useRouter } from 'vue-router'

// 创建路由实例
const router = useRouter()

// 返回门户页面方法
const goBackToPortal = () => {
  router.push('/')
}
```

#### 样式设计
```css
/* 返回门户按钮容器 */
.back-to-portal {
  position: absolute;
  bottom: 20px;
  left: 50%;
  transform: translateX(-50%);
  z-index: 100;
}

/* 按钮样式 */
.portal-btn {
  background: linear-gradient(135deg, #409EFF 0%, #66b1ff 100%) !important;
  border: none !important;
  box-shadow: 0 4px 12px rgba(64, 158, 255, 0.3);
  transition: all 0.3s ease;
}

/* 悬停效果 */
.portal-btn:hover {
  transform: scale(1.1);
  box-shadow: 0 6px 16px rgba(64, 158, 255, 0.4);
}

/* 点击效果 */
.portal-btn:active {
  transform: scale(0.95);
}

/* 收缩状态适配 */
.sidebar.collapsed .back-to-portal {
  left: 50%;
}
```

## 使用场景

### 场景1：从工作区返回门户
1. 用户在代码Review工作区完成任务
2. 点击左下角的返回按钮
3. 页面跳转到产品门户页面
4. 可以选择其他产品或重新进入工作区

### 场景2：侧边栏收缩状态
1. 用户收缩侧边栏以获得更大工作空间
2. 返回按钮自动调整位置，保持在收缩后的侧边栏底部中央
3. 功能正常可用

## 导航流程

```
门户页面 (/)
    ↓ 点击"AI代码Review助手"
工作区页面 (/workspace)
    ↓ 点击"返回门户"按钮
门户页面 (/)
```

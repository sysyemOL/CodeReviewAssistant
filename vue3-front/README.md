# AI代码Review助手 - 前端

## 项目说明

这是AI代码Review助手的前端部分，采用现代化的Vue 3技术栈构建。

## 目录说明

```
vue3-front/
└── vue-project/          # Vue 3前端项目
    ├── src/              # 源代码
    ├── public/           # 静态资源
    └── package.json      # 依赖配置
```

## 快速开始

```bash
# 进入项目目录
cd vue-project

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 🚀 启动前端项目

### 方式一：使用命令行

```powershell
# 1. 进入前端项目目录
cd vue3-front\vue-project

# 2. 确认依赖已安装（如果还没运行过npm install）
npm install

# 3. 启动开发服务器
npm run dev
```

### 方式二：使用VSCode

1. 在VSCode中打开集成终端（Ctrl + `）
2. 输入以下命令：
   ```bash
   cd vue3-front/vue-project
   npm run dev
   ```

### 访问应用

启动成功后，在浏览器中访问：
- **开发地址**: http://localhost:5173
- **网络地址**: http://你的IP:5173

## 📦 已完成的功能

### ✅ 核心功能

1. **三栏布局界面**
   - 左侧：历史会话列表（可折叠）
   - 中间：对话交互区
   - 右侧：Monaco代码编辑器

2. **会话管理**
   - 创建新会话
   - 删除会话
   - 重命名会话
   - 搜索会话

3. **消息展示**
   - Markdown渲染
   - 代码语法高亮
   - 复制消息
   - 时间显示

4. **文件上传**
   - 点击上传
   - 拖拽上传
   - 文件类型验证（支持.py, .js, .ts等）
   - 文件大小限制（最大10MB）

5. **代码编辑器**
   - Monaco Editor集成
   - 多语言语法高亮
   - 代码折叠
   - 多文件Tab切换

### ✅ 技术实现

- **框架**: Vue 3 (Composition API)
- **UI库**: Element Plus
- **状态管理**: Pinia
- **路由**: Vue Router
- **编辑器**: Monaco Editor
- **Markdown**: Marked + Highlight.js

## 💡 测试建议

- 左侧有一个"新对话"按钮

4. 上传后文件会显示在右侧编辑器

### 4. 测试代码编辑器

1. 上传代码文件后
2. 右侧会显示Monaco编辑器
3. 可以编辑代码、折叠代码块
4. 多个文件会显示Tab切换

## 🐛 故障排除

### 1. npm install 失败？

```powershell
# 清除缓存后重试
npm cache clean --force
npm install
```

### 2. 端口5173已被占用？

修改 `vite.config.js` 中的端口：
```javascript
server: {
  port: 5174,  // 改为其他端口
  // ...
}
```

### 3. Monaco Editor 加载慢？

这是正常的，Monaco Editor约2-3MB，首次加载需要一些时间。

### 4. Element Plus 图标不显示？

确保在 `main.js` 中正确注册了图标（已完成）。
